Introducción a HTML

HTML (Hypertext Markup Language) o lenguaje de marcado de hipertexto, utiliza un sistema de marcado compuesto de elementos que representan contenido. El marcado significa que con HTML declaras lo que se presenta a un espectador, no cómo se presenta. Las representaciones visuales se definen mediante hojas de estilo en cascada (CSS) y se muestra a través de los navegadores.

HTML a veces se denomina lenguaje de programación, pero no tiene lógica, por lo que es un lenguaje de marcado. Las etiquetas de HTML proporcionan significado semántico y legibilidad al contenido de la página.

Un elemento generalmente consta de una etiqueta de apertura (<nombre_elemento>), una etiqueta de cierre (</nombre_elemento>), que contienen el nombre del elemento rodeado por corchetes angulares y el contenido entre ellos:

<nombre_elemento> ... contenido ... </nombre_elemento>

Hay algunos elementos HTML que no tienen etiqueta de cierre ni contenido. Estos se denominan elementos vacíos. Algunos de estos elementos vacíos son:

  • <img>
  • <meta>
  • <link>
  • <input>
  • <br>

Los nombres de los elementos se pueden considerar como palabras clave descriptivas del contenido que contienen, como video, audio, tabla, pie de página, etc

Una página HTML puede constar potencialmente de cientos de elementos que luego son leídos e interpretados por un navegador web, y luego, se convierte en contenido audible o legible por humanos en la pantalla. Para este documento, es importante tener en cuenta la diferencia entre elementos y etiquetas:

Elementos: video, audio, tabla, pie de página
Etiquetas: <video>, <audio>, <table>, <footer>, </html>, </body>

Etiquetas de HTML

Analicemos una etiqueta ...

La etiqueta <p> representa un párrafo común. Los elementos suelen tener una etiqueta de apertura y una etiqueta de cierre. La etiqueta de apertura contiene el nombre del elemento en corchetes angulares (<p>). La etiqueta de cierre es idéntica a la etiqueta de apertura con la adición de una barra inclinada (/) entre los corchetes de apertura y el nombre del elemento (</p>).
El contenido puede ir entonces entre estas dos etiquetas: <p> Este es un párrafo simple. </p>.

Encabezados de HTML

HTML proporciona no solo etiquetas de párrafo simples, sino seis etiquetas de encabezado separadas para indicar encabezados de varios tamaños y espesores. Enumerado como el título 1 al título 6, el título 1 tiene el texto más grande y grueso mientras que el título 6 es el más pequeño y delgado, hasta el nivel de párrafo. Este tema detalla el uso adecuado de estas etiquetas.

Uso de títulos
Los encabezados se pueden utilizar para describir el tema que preceden y se definen con las etiquetas <h1> a <h6>. Por ejemplo:

<h1> Título 1 </h1>
<h2> Título 2 </h2>
<h3> Título 3 </h3>
<h4> Título 4 </h4>
<h5> Título 5 </h5>
<h6> Título 6 </h6>
Listas de HTML

HTML ofrece tres formas de especificar listas: listas ordenadas, listas desordenadas y listas de descripción. El uso de listas ordenadas es para secuencias ordinales para indicar el orden de los elementos de la lista, las listas desordenadas usan un símbolo definido, como una viñeta para enumerar elementos sin un orden designado, y las listas de descripción usan sangrías para enumerar elementos con sus hijos. Este tema explica la implementación y combinación de estas listas en el marcado HTML.

Lista ordenada
Se puede crear una lista ordenada con la etiqueta <ol> y cada elemento de la lista se puede crear con la etiqueta <li> como en el ejemplo a continuación:
<ol>
<li> Elemento </li>
<li> Otro artículo </li>
<li> Otro artículo más </li>
</ol>

Lista desordenada
Se puede crear una lista desordenada con la etiqueta <ul> y cada elemento de la lista se puede crear con la etiqueta <li como se muestra en el siguiente ejemplo:
<ul>
<li> Elemento </li>
<li> Otro artículo </li>
<li> Otro artículo más </li>
</ul>

Tablas de HTML

El elemento HTML <table> permite a los autores web mostrar datos tabulares (como texto, imágenes, enlaces, otras tablas, etc.) en una tabla bidimensional con filas y columnas de celdas. Por ejemplo, la siguiente tabla simple:
<tabla>
<tr>
<th> Encabezado 1 / Columna 1 </th>
<th> Encabezado 2 / Columna 2 </th>
</tr>
<tr>
<td> Fila 1 Columna de datos 1 </td>
<td> Columna 2 de datos de la fila 1 </td>
</tr>
<tr>
<td> Columna de datos 1 de la fila 2 </td>
<td> Columna 2 de datos de la fila 2 </td>
</tr>
</table>

Esto generará una tabla que consta de tres filas en total (<tr>): una fila de celdas de encabezado (<th>) y dos filas de celdas de contenido (<td>). Los elementos <th> son encabezados tabulares y los elementos <td> son datos tabulares. Puedes poner lo que sea desea dentro de un <td> o <th>.
